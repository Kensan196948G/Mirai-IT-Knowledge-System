# MVP定義 - Mirai IT Knowledge System

## 概要

情シス担当者の業務効率化と24時間対応可能なITサポート体制を構築するAIナレッジシステム

## MVP目標

### 主要機能

1. **AI対話ナレッジ作成** ✅ 実装完了
   - 自然言語での対話によるナレッジ自動生成
   - ITSM分類の自動判定
   - 任意のシステム名認識（AI駆動）

2. **AIインテリジェント検索** ✅ 実装完了
   - 自然言語クエリ対応
   - 根拠付き回答生成
   - 関連ナレッジ提案

3. **マルチAIエージェント統合** ✅ 実装完了
   - Claude: 統合・最終回答生成（プライマリ）
   - Gemini: 情報収集・調査
   - Perplexity: 根拠生成・エビデンス収集

> **注**: OpenAI APIは不使用（ChatGPT Plus契約と独立運用）

## 技術要件

### クロスプラットフォーム対応
- Ubuntu Linux（プライマリ）
- Windows 11（セカンダリ）

### 設計方針
- CLI中心設計（GUI依存なし）
- REST API提供
- WebUI（補助）

### AI構成
| AI | モデル | 役割 |
|----|--------|------|
| Claude | claude-sonnet-4-20250514 | 統合・回答生成 |
| Gemini | gemini-2.0-flash | 情報収集・調査 |
| Perplexity | sonar | 根拠収集 |
